Detailed Description

Sets up and controls two DC Motors using Timer A1 for PWM and 4 GPIO pins.

setDCMotorDirection() will set the motors to forward, reverse, or brake.

setDCMotorSpeed() will set the speed of the motors.

delay() is used for idling CPU and is for debugging purposes.

Control module for UART commands. Intialization method is provided, and must be ran before using the other methods.

uart_send() sends a character over the UART, but isn't interrupt safe.

uart_read() reads the UART Rx Buffer and returns the value on it.

uart_wasRead() returns whether or not the Rx buffer was read successfully. This is used for debugging purposes.

void setDCMotorDirection ( uint8_t  dir)

Sets the direction of the motors. If a 2 is received, they are set to go forward. If a 1 is received, they are set to go backwards. If a 0 is received, they are in brake mode.

Parameters
